# osint-framework
## Siklus Intelijen (The Intelligence Cycle)
Setiap kategori yang saya berikan adalah target dalam fase Collection (Pengumpulan). Namun, fundamental yang sebenarnya adalah bagaimana kategori tersebut masuk ke dalam siklus:
1.Planning/Direction: Menentukan target (Individu/Org/Infrastruktur).
2.Collection: Mengambil data mentah dari 6 kategori tersebut.
3.Processing: Mengonversi data mentah (misalnya: koordinat GPS menjadi alamat fisik).
4.Analysis/Production: Menghubungkan titik-titik antar kategori.
