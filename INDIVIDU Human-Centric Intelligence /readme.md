*Fokus pada identifikasi, pelacakan, dan profil psikologis serta digital subjek manusia.*

- [ ] **Identitas Inti (PII - Personally Identifiable Information)**
    - Nama Lengkap (Legal, Alias, Nama Gadis).
    - Tanggal dan Tempat Lahir (POB/DOB).
    - Nomor Identitas (NIK, SSN, Paspor, SIM).
    - Alamat Fisik (Rumah saat ini, Riwayat tempat tinggal).
- [ ] **Jejak Komunikasi Digital**
    - Alamat Email (Pribadi, Kerja, Sekunder).
    - Nomor Telepon (Mobile, Landline, VoIP).
    - Username Unik (Handle media sosial yang konsisten).
    - ID Aplikasi (Telegram UID, Discord ID, WhatsApp).
- [ ] **Social Media Intelligence (SOCMINT)**
    - Profil Platform (FB, X, IG, LinkedIn, TikTok, VK).
    - Analisis Jaringan (Daftar Teman, Pengikut, Interaksi).
    - Metadata Media (Geotag foto, EXIF data, Stempel waktu posting).
- [ ] **Latar Belakang Profesional & Finansial**
    - Riwayat Pendidikan (Sekolah, Universitas, Kursus).
    - Riwayat Karir (Jabatan, Perusahaan, Deskripsi Pekerjaan).
    - Status Finansial (Kepemilikan Aset, Catatan Kredit, Pailit).
    - Catatan Hukum (Kasus Pidana, Perdata, Tilang).
